# HR People Analytics: Employee Attrition & Workforce Insights

## Project Overview
This project explores workforce patterns using the IBM HR Analytics dataset 
(1,470 employees). The goal is to identify key drivers of employee attrition, 
analyse income distribution across roles, and surface actionable insights 
for HR decision-making.

This project demonstrates applied skills in Python, SQL, and data 
visualisation — relevant to People Analytics and HR Data roles.

---

## Key Questions Answered
- What is the overall attrition rate, and which departments are most affected?
- How does job satisfaction relate to employee turnover?
- Does tenure influence the likelihood of attrition?
- How does monthly income vary across job roles?

---

## Tools Used
| Tool | Purpose |
|------|---------|
| Python (pandas, matplotlib, seaborn) | Data cleaning, analysis, visualisation |
| SQL (SQLite) | Aggregation queries and workforce segmentation |
| Power BI | Interactive dashboard |

---

## Key Findings
- Overall attrition rate: **16.1%**
- Highest attrition in **Sales** department
- Employees with **0–2 years tenure** are most likely to leave
- Low job satisfaction (score 1) correlates strongly with attrition
